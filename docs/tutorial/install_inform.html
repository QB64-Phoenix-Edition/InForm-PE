Installing InForm-PE

InForm-PE serves as a graphical user interface (GUI) engine and WYSIWYG interface designer specifically tailored for QB64-PE. It is a modified version of InForm, designed to operate independently without relying on falcon.h dependencies. Therefore, using InForm-PE requires QB64-PE version 3.8.0 or newer. This modification also ensures that the entire InForm-PE library is now written in pure QB64-PE code.

Prelimanary

The following uses the top-level folder qb64-inform created when you followed the QB64pe installation.
Alternativly, you can substitute this for your top-level folder name.

Note: At the time of writing, the latest version of InForm-PE is v3.1.5.4

Download source files

Option 1:

Use the direct link to download the file. Save the downloaded file InForm-PE-master.zip to the folder qb64-inform.

Option 2:

If the above link does not work, follow these steps:

  1. Go to the Github folder: https://github.com/QB64-Phoenix-Edition/InForm-PE.
  2. Click on the green button labeled "Code" at the top right.
  3. Select the last option Download ZIP to download the file InForm-PE-master.zip.
  4. Copy this downloaded file to the folder named qb64-inform that you created during the QB64pe installation.

Extract files

To extract the source files from the downloaded file, follow these steps:

  1. Assuming you have 7-Zip installed, right-click on the file InForm-PE-master.zip.
  2. Mouse over the menu item 7-Zip, and from the side menu, click on Extract Here.
  3. A new folder named InForm-PE-master will be created, containing all the InForm-PE files.
  4. Optionally, to save space, you can delete the file "InForm-PE-master.zip" as it is no longer needed.

Note 1:
If you do not have 7-Zip installed on your computer, you can download and install it from the 7-Zip website select the appropriate version for your system. Alternatively, you can use a portable version of 7-Zip.

Portable 7-Zip

Instead of installing 7-Zip, you can use the portable version available from Portable Apps: https://portableapps.com/apps/utilities/7-zip_portable

7-Zip Portable details:

  1. Download the file 7-ZipPortable_23.01.paf.exe and save it to any folder, for example, 7Z-portable.
  2. Run the extraction process by double-clicking on the file 7-ZipPortable_23.01.paf.exe.
  3. During the process, when prompted, click on Run, OK, Next, Install, and finally Finish.
  4. Navigate into the extracted folder 7-ZipPortable and run 7-ZipPortable.exe by double-clicking on the file.
  5. The 7-Zip window will open. Under the name, double-click on "computer" to expand the folder and file structure.
  6. Navigate into folder qb64-inform and highlight (click) file InForm-PE-master.zip
  7. Click on File at the top left, then hover over the item 7-Zip, and from the side menu, select Extract Here.
  8. A new folder named InForm-PE-master will be created, containing all the InForm-PE files.
  9. "If you wish to save space, feel free to delete the file 'InForm-PE-master.zip' as it is no longer needed.

Compile InForm-PE

Before installing InForm-PE, you'll need to compile the source code following the detailed steps below.

  1. Navigate into the InForm-PE-master folder and execute one of the following scripts based on your operating system:
    • For Linux: run setup_inform_lnx.sh
    • For macOS: run setup_inform_osx.command
    • For Windows: double click on setup_inform_win.cmd
  2. After running the script, the compiler will execute, and after a short time, two files will be created:
    • InForm-PE-master\UiEditor.exe
    • InForm-PE-master\InForm\UiEditorPreview.exe

Note 1: Expect the following compilation time warnings: inability to find the aforementioned two files and warnings related to unused variables. Refer to the image on the right for more details.

Note 2: Warnings concerning unused variables may occur due to the use of id& AS LONG as a parameter in most event-handling routines, which may not always be utilized. It is generally safe to ignore these warnings.

Install InForm-PE

The final step involves installing InForm-PE, or more precisely, copying and moving the necessary folders and files as outlined below:

  1. Navigate into the InForm-PE-master folder.
  2. Copy the Inform folder and the file UiEditor.exe into the qb64pe folder.
  3. Optionally, to save space, you can delete the InForm-PE-master folder as it is no longer needed. However, it contains some examples, so you may want to keep these!

Test installation

To test the installation, navigate into your QB64 folder (qb64pe) and double-click the UiEditor.exe. file. This action will launch the InForm application, and you can expect to see the desired result as shown in the image on the right.

Please take note: Windows Firewall has detected that some features of this application have been blocked. To address this, follow the steps below:

  • Check the option labeled 'Private network' to allow the application access on private networks.
  • Uncheck the option labeled 'Public network' if it's checked, as it restricts access on public networks.
  • Finally, click on Allow access to grant the necessary permissions to the application.
